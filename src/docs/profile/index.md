# 自己紹介 

## 概要

土屋俊介 (@shunsock) はデータサイエンス領域に強みを持つソフトウェアエンジニアです。

データの転送・加工、データ解析・モデリング、ウェブアプリケーションやCLI、BIによる機能提供まで一連のタスクが可能です。

### インフラストラクチャ

[![](https://skillicons.dev/icons?i=aws,gcp,docker,terraform,nix)](https://skillicons.dev)
